# Grunge Meme

Bold, high-contrast design inspired by Russian YouTube thumbnails and internet meme culture. Combines distressed textures, aggressive typography, and checklist-driven layouts for maximum attention and relatability.

## Philosophy

This design embraces the raw, unpolished aesthetic of viral content. It prioritizes **immediate impact** over refinement, using grunge textures, bold contrast, and meme-style visuals to create an approachable, internet-native experience. Perfect for content platforms, tutorials, and community-driven applications.

## Typography

### Font Stack
- **Primary**: `'Impact', 'Franklin Gothic Bold', 'Arial Black', sans-serif` - Bold, attention-grabbing

- **Caption**: 14px / 400 / 1.4 / 0 - Small text

## Components

- **Success**: Green background (#4A7C4A), white text, 3px dark border
- **Hover**: Scale 1.05, add texture overlay, 0.2s transform
- **Disabled**: 50% opacity, no hover effect

## Layout Principles

- **Maximum Contrast**: Always pair light backgrounds with dark text/borders
- **Texture Everywhere**: Apply subtle grunge overlays to all surfaces
- **No Rounded Corners**: Keep everything sharp and rectangular
- **Bold Borders**: 3-4px borders on all interactive elements
- **Checklist Layout**: Use checkbox-style bullets for lists
- **Meme Elements**: Embrace internet culture (emojis, casual language, reaction images)

## Animation & Interaction

- **Hover Transforms**: Scale, rotate, translate (no smooth fades)
- **Timing**: 0.2s for all interactions (fast and punchy)
- **No Subtlety**: Make hover states obvious
- **Texture Shifts**: Increase texture opacity on hover
- **Shadows**: Use hard-edged shadows (no blur) on elevation
